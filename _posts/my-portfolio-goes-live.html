THE POWER OF 3

My new branding focuses on the humble equilateral triangle. What's great about this is the easy was to embed all manor of triangle based icons and logos by simply copying the vector graphics from Illustrator and pasting them straight into Sublime (my text editor of choice) as embedded SVGs. There's also the trick of making a full triangle with nothing more than some clever borders in CSS. I'm a huge fan of simplicity and cohesive branding staples throughout a website, so after a lot of thought around logos and testing with different ideas, I believe the triangle works very well in my new site.

BUILD IT

The power of Grunt as a task runner has been improving my workflow for quite a while now. For version control I'm using Github, and this combined with a relatively new service that I've found to work incredibly well is dploy.io.

This is a deployment service that links to Github (among other services) and will sync repositories with live and development web servers. This means that a simple push to my Github repo, and a few seconds later those changes are live on my website. What luxury. It's eradicated my need for any kind of file management on my web server as live and dev instances of my site are tied completely to Github branches. For one repository, Dploy is completely free - go check it out.
